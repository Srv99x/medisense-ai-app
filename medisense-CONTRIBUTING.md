# Contributing to MediSense AI

Thanks for your interest in contributing to MediSense! This is a full-stack AI health insights app — contributions across the frontend, backend, and documentation are all welcome.

---

## 🚀 Getting Started

1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/medisense-ai-app.git
   cd medisense-ai-app
   ```
3. **Create a branch** for your change:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Set up** the project (see [README.md](README.md#getting-started))
5. **Make your changes**, then commit and push
6. **Open a Pull Request** against the `main` branch

---

## 🧭 Areas to Contribute

| Area | Ideas |
|------|-------|
| 🧠 **AI / Prompts** | Improve the Gemini prompt for more accurate structured output |
| 🎨 **Frontend UI** | Add animations, improve mobile layout, dark mode polish |
| 📊 **Results Display** | Better charts or visual representations of probabilities |
| 🌍 **Multilingual** | Add support for symptoms input in other languages |
| 🧪 **Testing** | Add frontend unit tests (Vitest) or backend tests (pytest) |
| 📚 **Docs** | Screenshots, GIFs, improve setup instructions |
| ♿ **Accessibility** | Improve ARIA labels, keyboard navigation, contrast ratios |

---

## 📋 Pull Request Guidelines

- One feature or fix per PR — keep it focused
- Use clear commit messages:
  - `feat: add multilingual symptom input`
  - `fix: correct urgency level color on mobile`
  - `docs: add screenshots to README`
- If changing the Gemini prompt, include before/after example outputs
- If changing the UI, include a screenshot in the PR description

---

## ⚠️ Important: Medical Content Guidelines

This app deals with health information. When contributing:

- **Never** remove or weaken the medical disclaimer
- **Never** make the app sound like it can diagnose conditions
- All AI-generated content must remain clearly framed as informational/educational only
- If in doubt, err on the side of more caution, not less

---

## 🐛 Reporting Bugs

Open an [Issue](https://github.com/Srv99x/medisense-ai-app/issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your browser/OS and Node/Python versions

---

## 💡 Feature Requests

Open an [Issue](https://github.com/Srv99x/medisense-ai-app/issues) with the `enhancement` label. Describe the problem it solves and your proposed approach.

---

## 🤝 Code of Conduct

Be respectful, constructive, and patient. All skill levels are welcome — this is a learning project.
